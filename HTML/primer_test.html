Lista de las etiquetas HTML más comunes y utilizadas:


Etiquetas de estructura básica:
html: Define un documento HTML.
head: Contiene metadatos y enlaces a recursos externos.
title: Define el título del documento.
body: Contiene el contenido visible del documento.
Etiquetas de encabezado:
h1 a h6: Define encabezados de nivel 1 a 6, siendo h1 el más importante.
header: Define la cabecera o encabezado de una sección o del documento completo.
Etiquetas de párrafo y texto:
p: Define un párrafo.
br: Inserta un salto de línea.
strong o b: Muestra el texto en negrita.
em o i: Muestra el texto en cursiva.
u: Subraya el texto.
s: Muestra el texto tachado.
Etiquetas de listas:
ul: Define una lista desordenada.
ol: Define una lista ordenada.
li: Define un elemento de lista.
Etiquetas de enlaces:
a: Define un enlace.
href: Especifica la URL de destino del enlace.
target: Define cómo se abrirá el enlace (por ejemplo, en una nueva ventana).
Etiquetas de imágenes:
img: Inserta una imagen.
src: Especifica la URL de la imagen.
alt: Proporciona un texto alternativo para la imagen (accesibilidad).
Etiquetas de tablas:
table: Define una tabla.
tr: Define una fila de la tabla.
td: Define una celda de datos.
th: Define una celda de encabezado.
Etiquetas de formularios:
form: Define un formulario.
input: Define un campo de entrada.
textarea: Define un área de texto multilínea.
select: Define un menú desplegable.
option: Define una opción en un menú desplegable.
button: Define un botón.
Etiquetas de estilo y presentación:
style: Define reglas de estilo CSS dentro del documento HTML.
link: Enlaza una hoja de estilo externa.
Atributos adicionales:
Atributos globales:
id: Identifica de forma única un elemento.
class: Asigna una o varias clases al elemento.
style: Define estilos CSS inline para el elemento.
Atributos de enlaces y recursos:
href: Especifica la URL de destino del enlace.
target: Define cómo se abrirá el enlace.
src: Especifica la URL de la fuente de un recurso (imagen, audio, video).
alt: Proporciona un texto alternativo para imágenes.
rel: Define la relación entre el documento actual y el recurso enlazado.
Atributos de formularios:
action: Especifica la URL del servidor que procesará los datos del formulario.
method: Define el método HTTP utilizado para enviar los datos del formulario (GET o POST).
name: Asigna un nombre al campo de entrada o formulario.
Atributos de tablas:
colspan: Define el número de columnas que debe abarcar una celda.
rowspan: Define el número de filas que debe abarcar una celda.
Etiquetas de encabezado:
header: Define la cabecera de una sección o del documento completo.
hgroup: Agrupa varios elementos de encabezado (h1-h6) juntos.
Etiquetas de sección:
section: Define una sección lógica o temática del documento.
article: Define un contenido independiente y autónomo dentro del documento.
aside: Define un contenido relacionado o adicional.
Etiquetas de detalles y resumen:
details: Define detalles adicionales o información oculta.
summary: Define el resumen o título de los detalles.
Etiquetas de marca semántica:
mark: Resalta o marca un texto específico.
time: Define una fecha o hora.
Etiquetas de representación de datos:
progress: Muestra el progreso de una tarea o proceso.
meter: Muestra una medida escalar dentro de un rango conocido.
Etiquetas de navegación:
nav: Define una sección de navegación.
Etiquetas de diálogo:
dialog: Define un cuadro de diálogo o ventana emergente.
Etiquetas de multimedia:
figure: Define una figura o contenido multimedia con una leyenda.
figcaption: Define la leyenda de una figura.
Etiquetas de contenido incrustado:
embed: Incrusta contenido externo, como multimedia, dentro del documento.
iframe: Incrusta un documento HTML externo dentro del documento actual.
Etiquetas de mapa de imagen:
map: Define un mapa de imagen para enlazar áreas específicas de una imagen.
area: Define una zona en un mapa de imagen.
Etiquetas de salida de datos:
output: Muestra el resultado de un cálculo o procesamiento.
